# ⚡ Guia Rápido - Trocar Conta GitHub Desktop

## 🎯 Objetivo
Garantir que o projeto RAVAR vá para a conta **RavatImoveis** (não outra).

---

## ✅ Solução em 3 Passos (2 minutos)

### **PASSO 1: Trocar Conta**

**Windows:**
1. GitHub Desktop → File → Options → Accounts
2. Clique "Sign Out"
3. Clique "Sign in"
4. Escolha "Sign in using your browser"
5. Faça login com **RavatImoveis**

**Mac:**
1. GitHub Desktop → Preferences → Accounts
2. Clique "Sign Out"
3. Clique "Sign in"
4. Escolha "Sign in using your browser"
5. Faça login com **RavatImoveis**

✅ **Agora o Desktop está na conta certa!**

---

### **PASSO 2: Se Já Publicou Errado**

**A. Delete o repositório da conta errada:**
1. Vá para GitHub.com (conta errada)
2. Abra o repositório
3. Settings → Danger Zone
4. "Delete this repository"
5. Confirme

**B. Remova do Desktop (NÃO deleta arquivos):**
1. GitHub Desktop
2. Repository → Remove
3. Confirme (seus arquivos ficam intactos)

---

### **PASSO 3: Publicar na Conta Certa**

1. GitHub Desktop → File → Add Local Repository
2. Selecione a pasta do projeto RAVAR
3. Clique "Create a Repository"
4. **IMPORTANTE:** Verifique que mostra "RavatImoveis" como owner
5. Nome: `ravar-imoveis`
6. Clique "Publish repository"

✅ **Pronto! Projeto na conta correta!**

---

## 🔍 Como Verificar se Está Certo

**No GitHub Desktop:**
- Canto inferior esquerdo deve mostrar: **"Signed in as RavatImoveis"**

**No GitHub.com:**
- Vá para: https://github.com/RavatImoveis
- Deve ver: `ravar-imoveis` na lista

---

## 🆘 Se Não Funcionar

**Reset total (5 minutos):**

1. Desinstale GitHub Desktop
2. Reinstale: https://desktop.github.com
3. Login com **RavatImoveis**
4. Add Local Repository
5. Publish
