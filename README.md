# 🏢 RAVAR - Imóveis de Luxo

Sistema completo de gestão de imóveis de alto padrão com design system RAVAR (Navy #0A1929 + Gold #AF9042).

## 🚀 Tecnologias

- **Frontend:** React + TypeScript + Tailwind CSS v4
- **Backend:** Supabase Edge Functions (Deno + Hono)
- **Database:** Supabase KV Store
- **Storage:** Supabase Storage (imagens)
- **Routing:** React Router v7
- **Deploy:** Vercel

## ✨ Funcionalidades

- 🏠 Homepage com Hero e imóveis em destaque
- 🔍 Sistema de filtros avançados (43 categorias em 4 colunas)
- 📱 Design responsivo e mobile-first
- 🎨 Design system completo com cores RAVAR
- 🖼️ Upload de múltiplas imagens por imóvel
- ⚙️ Painel Admin para gestão de imóveis
- 🔐 Sistema escalável com Supabase

## 📋 Páginas

- `/` - Homepage
- `/explorar` - Lista de imóveis com filtros
- `/imovel/:id` - Detalhes do imóvel
- `/anunciar` - Formulário para anunciar imóvel
- `/admin` - Painel administrativo
- `/setup` - Migração inicial de dados

## 🛠️ Desenvolvimento Local

```bash
# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev

# Build para produção
npm run build
```

## 🌐 Deploy

Este projeto está configurado para deploy automático na Vercel:

1. Conecte o repositório GitHub à Vercel
2. Configure as variáveis de ambiente do Supabase
3. Deploy automático a cada push
