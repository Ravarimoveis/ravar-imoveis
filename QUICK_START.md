# 🚀 Quick Start - Sistema RAVAR Supabase

## ⚡ 3 Passos para Começar

### **1️⃣ Migre os Dados (uma vez)**
Acesse: `https://seu-app.vercel.app/setup`
- Clique em "Iniciar Migração"
- Aguarde completar (9 imóveis migrados)

### **2️⃣ Gerencie Imóveis**
Acesse: `https://seu-app.vercel.app/admin`
- Adicionar novo imóvel: Botão "Novo Imóvel"
- Editar: Botão "Editar" no card
- Deletar: Botão "Deletar" no card

### **3️⃣ Upload de Fotos Reais**
No Admin > Editar RAVA-010:
1. Remova fotos do Unsplash (clique ❌)
2. Clique em "Upload de fotos"
3. Selecione suas 20 fotos reais
4. Aguarde upload automático
5. Clique "Atualizar Imóvel"

## 📝 Criar Novo Imóvel

**Dados obrigatórios:**
- ID (ex: RAVA-011)
- Título
- Tipo (Venda/Aluguel)
- Bairro
- Área (m²)
- Descrição

**Valores:**
- 0 = "Sob Consulta"
- Número = Exibe o valor

**Fotos:**
- Upload múltiplo (até 20 fotos)
- Máximo 10MB por foto
- Formatos: JPG, PNG, WEBP
